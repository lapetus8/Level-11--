外表与性格

原为雾隐“忍刀七人众”的成员之一，有着跟鲛鲨相像的面孔（两边的肩膀处也有鲨鱼鳃）、 肤色及尖锐的牙齿，金色瞳孔。死前为晓之成员，非常尊敬宇智波鼬，甚至会用敬语“鼬先 生”，还会关心鼬的身体，在晓之中算是很听从命令的一个，从不透漏关于自己的事。

身世与经历

回忆篇中鬼鲛他在成为叛忍前，是前任鲛肌主人西瓜山河豚鬼的亲信部下（当时的武器为刀），长年以来一直尽责地为雾忍者村执行难以想像的黑暗任务，其中甚至包括杀死一起执行任务的同伴，避免其落入敌方审问部队手中防泄漏情报，之后他杀死了通敌的上司西瓜山河豚鬼，成为鲛肌的新主人.
面对现身在自己面前的第四代水影矢仓，鬼鲛感叹自己只是个虚假的存在，突然有个自称宇智波斑的黑影从第四代水影背后出现，表示能够将鬼鲛从虚假的痛苦中解救出来，更邀请他成为自己的伙伴，鬼鲛确认了他的面容后，对他的“月之眼”计划颇为认同，之后加入了晓。

主要忍术

招式名称 简介 种类 等级
潜鲨突袭 潜入水中，再出现快速攻击敌人。 体A
通灵术 召唤兽是鲨鱼。 忍 C
土遁·土中潜航 使使用者周围的土地液化，能在土中游泳快速接近敌人。 忍 C
水遁·水分身术 利用查克拉控制水进而制造分身的忍术，对此分身的攻击都会被水化解。 忍 C
水遁·水牢术 利用查克拉制造水牢将对手困住的忍术，是常人100%不可能脱出的忍术。 忍 C
水遁·爆水冲波 一口气吐出大量的水，能将一大片低漥谷地瞬间变成湖泊。是需要庞大查克拉的大范围忍术。能将战斗场地瞬间变成适合鬼鲛战斗的水域环境。 忍 B
水遁·大爆水冲波 “水牢术”的升级版，制造一个足球场大小的水球把自己和敌人包在中间，水球会以自己为中心移动。因此只要自己在水中的速度比较快的话，对手就绝不可能逃出生天。是使用水牢鲛舞之前的前奏招式。 忍A
水遁·水鲛弹术 把水以鲨鱼的样子弹射出去，有很大的破坏力，鬼鲛也可进入水鲛弹中移动。 忍 B
水遁·五食鲛 由五根手指放出五只“水鲛弹”，连续不断地攻击对方，而且这些鲨鱼就算被击破，也可以在水境中再生。 忍 B
水遁·千食鲛 放出上千只“水鲛弹”，像海浪般攻击。 忍 B
水遁·大鲛弹术 鬼鲛第二四强之术，“水鲛弹”超级升级版，产生超巨大的鲨鱼之囗，就像一只巨鲨把眼前东西一囗噬掉，能吸收对方查克拉、遇强愈强。 忍 A
水遁·水牢鲛舞 先由鬼鲛使出“大爆水冲波”困住敌人，然后鬼鲛与大刀鲛肌合体形成鱼人模式向敌人发动进攻。鱼人模式下的鬼鲛可以用腮呼吸，可以通过查克拉来感知敌人的方位，敌人只要与鬼鲛近身接触就会被吸取查克拉，直至被鬼鲛将查克拉吸尽。 忍 S
